 ¿Que usamos para llegar a este punto?
 Instalamos Git, Github.CLI, VScode y SumatraPDF.
 Una vez instalados los dos primeros se deben configurar de manera predeterminada (ambos señalan las configuraciones)

 Una vez con todo listo deben crear un repositorio en GitHub
 Una vez creado se debe conectar con el repositorio en la tercera ventana en el menú izquierdo de la pantalla
 Una vez ahí, presionamos donde dice "Clonar repositorio" e iniciar sesión con nuestra cuenta de Github
 Una vez iniciado, editamos el texto dentro del README.md generaro presionar en guardar o Ctrl + S y presionamos 'Commit and Push'
 Una vez dentro te mostrara la terminal y ahi tendras que colocar los siguientes comandos
 git config --global user.name [Nombre de usuario]
 git config --global user.email [Correo@gmail.com]
